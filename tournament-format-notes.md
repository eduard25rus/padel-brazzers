# Tournament Format Notes

- `THE BEST MIDDLE (PM)` was a paired `Round Robin` PRO tournament.
- Future paired Round Robin tournaments should use the same structure: pair-based final standings, round selector, court results, winners highlighted in green, pair-rating badges, and standings after each round.
- `Americano Brazzers PRO` is an individual-score format: the public card should use personal standings, changing partners by round, pair-rating badges only inside round matches, and individual ratings in player badges.
